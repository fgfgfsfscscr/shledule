Настройки синхронизации

Для синхронизации между устройствами нужен GitHub токен и репозиторий:

  1. Создайте приватный репозиторий на GitHub
  2. Создайте Personal Access Token в GitHub Settings → Developer settings → Personal access tokens → Tokens (classic)
  3. Дайте токену права: repo (полный доступ к репозиториям)
